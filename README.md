# WebHook-Spammer
웹후크룰 이용해 반복적으로 메시지를 전송합니다.
## 설치법
```bash
git clone https://github.com/Auto-Discord/WebHook-Spammer.git
cd WebHook-Spammer
g++ -o main main.cpp
```

## 설정법
[config.h](config.h) 파일을 메모장, VIM 등으로 편집합니다. 관련 설명은 모두 이 파일에 들어 있습니다.

## 사용법
- Linux, MacOS
```bash
./main
```
- Windows
단순히 `main.exe` 파일을 실행해주세요.
